# mitbbs-stock-digest-site

Public site for daily stock forum digests from [newmitbbs.com](https://newmitbbs.com) 股海弄潮 (Stock) forum.

**Live at:** https://ixnix.github.io/mitbbs-stock-digest-site/

## Features

- Daily digests summarizing insightful stock trading discussions
- Weekly recaps with prediction verification against Yahoo Finance data
- RSS feed at [`/feed.xml`](https://ixnix.github.io/mitbbs-stock-digest-site/feed.xml)
- Organized by week (Sun–Sat) with holiday labels

## Tech

- Jekyll + GitHub Pages (auto-deploys on push)
- Minima theme with custom layouts
- Content generated by automated scraper + Claude AI summarization
